Request Management Help

Understanding Requests

Request management allows employees to submit various types of requests that require approval from managers or administrators.

Types of Requests:
  • Leave Requests: Time off for vacation, sick days, or personal matters
  • Overtime Requests: Additional working hours beyond regular schedule
  • Equipment Requests: Requests for new or replacement equipment
  • Other Requests: Any other type of request that requires approval

Request Statuses

Pending

Request has been submitted but not yet reviewed.

Approved

Request has been reviewed and approved.

Denied

Request has been reviewed and denied.
